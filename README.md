# MERN EMPLOYEE SALARY MANAGEMENT { SiPeKa }

## Why this HRMS?
I chose the MERN EMPLOYEE SALARY MANAGEMENT { SiPeKa } project because I "Wanted to maintain the Referenced workflow" and build upon a solid, existing React/Node.js foundation for employee salary and data management.

## Setup Instructions
In order to run this project locally:

1. Clone this repository to your local machine.
2. Ensure you have MySQL installed and running. Create a new MySQL database.
3. Split your terminal into two (one for the Backend, one for the Frontend).

### Backend Setup
1. Navigate to the Backend folder:
   ```bash
   cd Backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your `.env` file in the Backend directory:
   ```env
   APP_PORT=5000
   SESS_SECRET=your_secret_key
   ```
4. Start the backend server:
   ```bash
   npm start
   ```

### Frontend Setup
1. Navigate to the Frontend folder:
   ```bash
   cd Frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the React app:
   ```bash
   npm run dev
   ```
